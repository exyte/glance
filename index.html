Overview and features

Glance is an Eclipse plug-in providing incremental text search in Eclipse. Simple. Powerful. Free.

Download

Just drag and drop following icon into a running Eclipse workspace to install Glance

You can also clone the project with Git by running: 

$ git clone git://github.com/ystrot/glance

Usage

  • Move focus to component where you want to search
  • Open Glance using Ctrl + Cmd + F shortcut on Mac or Ctrl + Alt + F on other platforms
  • Enter text you want to search
  • Use Enter or Shift + Enter to find next/previous match
  • Close search box using Esc

Feel free to report any issue or request an improvement to our bug tracker
